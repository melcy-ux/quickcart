Reflection

Setting up XAMPP went fairly smoothly overall, but I ran into one issue early on — port 80 was already occupied by another process on my machine, which stopped Apache from starting. Creating my first PHP file felt different from anything I had done in HTML and CSS; understanding that the server processes the script before the output reaches the browser was a key shift in how I thought about things. Connecting to quickcart_db using mysqli_connect() actually worked on the first attempt, which was a good way to start the project.

Reflection

Building the login system was more involved than the earlier weeks. The most frustrating problem was with sessions not persisting — after logging in, users were being sent straight back to the login page on every subsequent request. After spending time with the problem I realized I had not placed session_start() at the very top of each protected page, meaning the session variables were simply not available when the guard check ran. Once that was in place the redirects worked as expected. I also chose to use password_hash() and password_verify() rather than storing passwords as plain text, which added a small amount of work to both the registration and login scripts but felt like the correct approach.

Reflection

Designing the database schema required the most thought of any week so far. Rather than storing the category name as a text column in every product row, I created a separate categories table and linked it to products using a foreign key on category_id. This caused a problem early on because phpMyAdmin threw an error when I tried to insert products before the corresponding categories existed — once I understood that foreign key constraints enforce the insertion order, it made complete sense. Writing the JOIN query to combine both tables was unfamiliar at first and took a couple of attempts to get the syntax correct. When the products page finally loaded with proper category names displayed under each item, the whole project started to feel like a real working website.
